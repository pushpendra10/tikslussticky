Tikslus Sticky

Tikslus Sticky allows any HTML content to stick to screen when user scrolls down

You can see live examples on this page

Include jquery,tikslussticky.js files

Example 1: Simple text

Javascript

		
                    $.fn.tikslussticky({
                    content:'hello world !',
                    customID:'hello',
                    stickAt:'center-left'
                    
                    });

Example 2: User Registration Link

Javascript

		
                    $.fn.tikslussticky({
                    content:"
Sign Up now !
",
                    customID:'signup',
                    stickAt:'center-right'
                    
                    });

Example 3: Print command

Javascript

		
                    $.fn.tikslussticky({
                    content:"
PRINT
",
                    customID:'signup',

                    });

                    });
